## Project Title

Student Grade Management System

## Project Overview

This project was developed as part of the AI Governance & Digital Safety Bootcamp Week 1 assignment. The goal of the project is to demonstrate an understanding of Python fundamentals, including variables, data types, functions, control structures, and error handling.

The Student Grade Management System allows users to create student records, enter grades, calculate average scores, determine letter grades, and identify whether a student is passing or failing. The program uses different Python data types and programming concepts to manage and display student information effectively.

---

## Features

- Add student records
- Store and manage student grades
- Calculate average scores automatically
- Assign letter grades based on performance
- Display pass/fail status
- View all stored student records
- Handle invalid user input using error handling

## Sample Functions Used

### Calculate Average

Calculates the average of all grades entered for a student.

### Determine Letter Grade

Assigns a letter grade based on the student's average score.

| Average Score | Grade |
|--------------|-------|
| 90 - 100 | A |
| 80 - 89 | B |
| 70 - 79 | C |
| 60 - 69 | D |
| Below 60 | F |

### Create Student Record

Creates a student record using:

- Student ID
- Student Name
- Grades
- Average Score
- Letter Grade
- Passing Status
